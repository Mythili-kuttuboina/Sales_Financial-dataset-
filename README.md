
#Sales_Financial_dataset

The Interactive Business Dashboard is designed to provide stakeholders with a comprehensive and real-time view of business performance. It transforms raw data into actionable insights through intuitive and interactive visualizations.

This dashboard helps decision-makers monitor key metrics such as sales, customer behavior, and product performance, enabling faster and more informed business decisions.

🎯 Objectives
Deliver a centralized view of key business metrics
Analyze sales trends and revenue performance
Identify top-performing products and categories
Understand customer purchasing behavior
Enable interactive exploration of data

🛠️ Tools & Technologies
Power BI / Tableau
Excel / CSV Dataset
SQL (for data extraction and transformation)
Power Query (Data Cleaning & Transformation)
📌 Dashboard Features
📍 KPI Metrics
Total Sales
Revenue
Customer Count
Average Order Value

📊 Visualizations
Bar Chart → Product-wise sales performance
Line Chart → Sales trends over time
Pie/Donut Chart → Category distribution
Stacked Charts → Comparative analysis
🎛️ Interactivity
Slicers for filtering by:
Year
Product Category
Customer Segment
Drill-down capability for deeper insights
Cross-filtering across all visuals
